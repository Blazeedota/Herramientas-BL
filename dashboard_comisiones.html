Configuración API & Consulta Individual

Ingresa tu API key y selecciona un RUC para consultar la actividad económica SUNAT.
Endpoint: https://api.apis.net.pe/v2/sunat/ruc?numero={RUC}  ·  Header: Authorization: Bearer <token>
Devuelve: razón social · tipo contribuyente · actividad económica (CIIU) · estado · condición · dirección · fecha inscripción.
También compatible con API oficial SUNAT (api.sunat.gob.pe) con credenciales OAuth propias.
